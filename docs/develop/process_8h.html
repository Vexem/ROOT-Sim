#define lvt (   lid)    (LPS(lid)->bound != NULL ? LPS(lid)->bound->timestamp : 0.0)

This macro retrieves the LVT for the current LP. There is a small interval window where the value returned is the one of the next event to be processed. In particular, this happens during the scheduling, when the bound is advanced to the next event to be processed, just before its actual execution.
